Objective

This project demonstrates how caching improves system performance by reducing database access time. It implements an LRU (Least Recently Used) Cache in C++ integrated with SQLite, and provides a dashboard to visualize cache efficiency using metrics like hit rate, misses, and response trends. The goal is to show how real-world systems optimize speed and scalability using caching techniques.

Usage

Select a key from the dropdown and click Fetch to retrieve data from cache/database. Add new items using the input field, update or delete selected entries, and observe cache performance through hit rate and real-time chart updates.

Highlights

• LRU Cache implementation in C++
• SQLite database integration
• Real-time cache hit/miss tracking
• Interactive performance visualization
• REST API-based communication
